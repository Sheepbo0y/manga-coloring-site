## 漫画上色网站 - 自动迭代任务

### 项目位置
~/projects/manga-coloring-site/

### 迭代流程
1. 读取 ITERATION_PLAN.md 了解当前进度
2. 检查上次迭代的完成情况
3. 根据优先级选择下一个任务
4. 执行代码修改
5. 测试验证
6. 提交到 GitHub

### 当前高优先级任务

1. **修复未登录用户404问题**
   - 创建 ProtectedRoute 组件
   - 未登录用户访问 /profile、/upload 等页面时自动跳转到 /login
   - 登录后自动跳转回原页面

2. **替换上色服务为 Seedream API**
   - 修改 backend/src/services/queue.ts
   - 使用 Seedream API 生成彩色版本
   - API Key: 从 MODELVERSE_API_KEY 环境变量获取

3. **前端动效增强**
   - 添加 Framer Motion 动画库
   - 首页 Hero 区域渐入动画
   - 作品卡片悬停缩放效果
   - 页面切换淡入淡出过渡

4. **其他待修复问题**

   - 图片懒加载
